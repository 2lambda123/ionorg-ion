## Features

- [x] Server
    - [x] OS
        - [x] CentOS 7
        - [x] macOS Mojave
    - [x] Signal
        - [x] WebSocket
    - [x] Media
        - [x] WebRTC
        - [x] RTP/RTCP
        - [x] Nack
        - [x] PLI
        - [x] Anti-Loss-Package 30%~50%
    - [x] Distributed System
        - [x] ION-ION RTP relay
        - [x] MQ support
- [x] Client
    - [x] SDK
        - [x] Flutter
        - [x] JS
    - [x] Demo

## How to use
### 1. make key
```
./scripts/makeKey.sh
```
### 2. build
```
#centos
./scripts/centos/installDeps.sh

#mac
./scripts/mac/installDeps.sh
```
### 3. run
```
#centos
./scripts/centos/allRestart.sh

#mac
./scripts/mac/allRestart.sh
```
### 4. let's chat
Open this url with chrome

```
https://yourip:8080
```
